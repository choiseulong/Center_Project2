  1. 색상 참고는 encycolorpedia 사이트를 이용했습니다. 특정 색상을 검색하면 어울리는 색? 정반대 색? 등을 보여줍니다. 사이트 바로가기

  3. 페이지 상단 좌측구석에 위치한 Likelion HYUERICA 7th은
    #mark{
    position: absolute;
    top : 5px;
    left : 10px; }
    처럼 해서 워터마크 느낌으로 두고 싶었어요! 다양한 방법들이 있을 것 같네요.

  4. 백그라운드 이미지를 두는 방법.
    body {
    background-image: url("");
    background-repeat: no-repeat;
    background-size: cover;
    background-position: center center;
    background-attachment: fixed; }
    바디 전체에 이미지를 입혔어요. 참고는 여기 클릭!

  5. Button 이나 Navs는 부트스트랩을 통해서 얻을 수 있는데 Textarea는 테마 구하기가 조금 더 번거로운 느낌은 저만 가지고 있었겠...죠?
    textarea {
    padding: .5em; 여백을 주는 건데 사실 placeholder 부분이 너무 경계에 딱 붙어서 뒀습니다.
    line-height : normal;
    border-radius: 3%; 테두리를 둥그~스름하게 만들어줍니다.
    border:2px solid #3e34d4; }
    textarea::placeholder{ 안내문? 같은 글귀를 꾸미는 방법! 이번과제하면서 ::로 스타일 입히는 것을 처음봤어요
    color:#3e34d4;
    font-size: 25px; }
